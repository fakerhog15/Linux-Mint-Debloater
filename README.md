# Linux Mint Debloater

Removes pre-installed bloatware from Linux Mint and optionally installs a lightweight, privacy-respecting browser.

## What it removes

Firefox, LibreOffice, Thunderbird, HexChat, Transmission, Rhythmbox, Celluloid, Hypnotix, Pix, Simple Scan, Drawing, Gnote, Xreader, Redshift, Baobab, Seahorse, Warpinator, Onboard, GNOME Calendar, GNOME Logs, GNOME Power Manager, Webapp Manager, and Thingy.

## What it offers

Install one of these as a replacement browser:

- **LibreWolf** – Privacy-focused Firefox fork (official repo)
- **Helium** – Lightweight privacy browser (apt repo)
- **Zen Browser** – Minimal browser (official install script, Terminal.sh only)
- **Brave Nightly** – Cutting-edge Brave (official repo)

## Usage

Run either script:

```bash
bash Terminal.sh    # terminal interface
bash GuiMenu.sh     # GUI interface (requires zenity)
```

Both let you select which packages to remove and optionally install a new browser.
