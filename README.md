Linux Mint Debloater
The Linux Mint Debloater is a set of scripts designed to streamline your Linux Mint experience by allowing you to easily remove pre-installed software and optionally install alternative, privacy-focused browsers.
Features
Selective Removal: Choose exactly which pre-installed applications you want to purge from your system.
User-Friendly Interfaces: Provides both a graphical interface (using Zenity) for ease of use and a terminal-based option for advanced users.
System Cleanup: Automatically performs autoremove and clean operations to ensure no unnecessary leftover files remain.
Browser Integration: Offers an integrated post-debloat step to install and set up alternative browsers, including:
1: Librewolf
2: Helium Browser
3: Brave Nightly
4: Zen Browser (Terminal version)
How It Works
The scripts target a pre-defined list of common packages, such as Firefox, LibreOffice, Thunderbird, and various GNOME/Mint utilities.
Selection: You select the packages you wish to remove from the provided list.
Removal: The script executes a purge command to remove the selected software and cleans up dependencies.
Browser Setup: Optionally, you can choose to add official repositories and install your preferred alternative browser to replace the removed defaults.
